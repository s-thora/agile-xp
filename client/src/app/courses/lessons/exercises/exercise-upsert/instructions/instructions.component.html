General: Files

Path

The files are placed in "files" directory.

 Black Box (with Files)

Description

Description should contain a number of bugs in the hidden source code. Also it should contain code structure or functions names, or any kind of information to give a student needed knowledge about the hidden code. So the student would be able to call necessary functions from his tests.

Hidden source code

Hidden source code is used for controlling, it should be compilable and contain several logical mistakes. It should be designed in a special way, so it would be used correctly.

The source code should initialise a switcher, which would turn on and off single bugs:

private BlackBoxSwitcher switcher = new BlackBoxSwitcher();
